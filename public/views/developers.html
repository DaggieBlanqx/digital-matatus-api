Welcome to the Developers API

Base url: http://digitalmatatus.herokuapp.com

Get information about routes: GET /api/v1/routes 
{
  "route_id": {
    "short_name": "route_short_name",
    "long_name": "route_long_name",
    "description": "route_desc"
  },
  ...
}
Get all positions of all routes: GET /api/v1/shapes 
{
  "shape_id": [
    [
      "shape_pt_sequence",
      "shape_pt_lat",
      "shape_pt_lon"
    ]
  ],
  ...
}
Match route ids to shape ids GET /api/v1/trips 
{
  "route_id": "shape_id",
  ...
}
