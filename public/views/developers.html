API for Nairobi bus route data

Base url: http://digitalmatatus.herokuapp.com

Get all routes: GET /api/v1/routes 
{
  "route_id": {
    "short_name": "route_short_name",
    "long_name": "route_long_name",
    "description": "route_desc"
  },
  ...
}
Get shape id of a specific route: GET /api/v1/routes/getShape/:route_id 
"shape_id"
Get route by route_id: GET /api/v1/routes/:route_id 
{
    "short_name": "route_short_name",
    "long_name": "route_long_name",
    "description": "route_desc"
}
Get shape by shape_id: GET /api/v1/shapes/:shape_id 
[
    [
      "shape_pt_sequence",
      "shape_pt_lat",
      "shape_pt_lon"
    ],
    ...
]
Get all shapes of all routes (677kB file, very slow!): GET /api/v1/shapes 
{
  "shape_id": [
    [
      "shape_pt_sequence",
      "shape_pt_lat",
      "shape_pt_lon"
    ]
  ],
  ...
}
Get all route ids matched to shape ids GET /api/v1/trips 
{
  "route_id": "shape_id",
  ...
}
